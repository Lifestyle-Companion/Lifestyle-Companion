DECISION LOG — ALPHA 1.1

- Rooms are always visible on the Home Circle.
- Features are activated only when the user chooses to use them.
- Room-specific information is requested conditionally, not during unrelated setup.
- Master profile information is entered once and reused by applicable Rooms.
- Every main Room navigation layout must remain a mathematically true circle.
- Core Room navigation must fit without vertical scrolling on supported phone and tablet screens.
- Coming-soon Rooms may remain visible at public launch, provided the active core experience is polished and satisfying.
- Interest in unfinished Rooms should be measured using more than raw taps: viewed, setup interest, setup completion and repeat usage.
- Lifestyle Companion should inform, encourage and assist without making users feel guilty.
