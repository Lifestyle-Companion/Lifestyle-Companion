LIFESTYLE COMPANION — FOUNDER'S TRIAL EDITION
Alpha 1 · Sprint 1 Foundation

WHAT THIS BUILD IS
This is the first working foundation build for the Founder's Trial Edition. It is designed for mobile phones, tablets and desktop browsers and can be hosted on GitHub Pages.

INCLUDED NOW
- New welcome-only Home page: no Life Pulse and no data scorecards.
- Centred Circle of Life with no intended scrolling on Home and Food & Nutrition entry pages.
- Up to 10 local tester profiles per device.
- Companion setup: user name, Companion name, 12 character choices, personality, voice and colour theme.
- Companion name and character can be changed later.
- Voice command framework using the browser's speech recognition where supported.
- Spoken replies using the device's available text-to-speech voices.
- Working voice navigation for Food & Nutrition, Settings, Home and jokes.
- Restored Activity Level, Weight Goal and Fasting Preference settings.
- Save-before-leaving prompt for unsaved settings.
- Installable Progressive Web App foundation and offline caching.

IMPORTANT LIMITATION
This Sprint 1 build stores data in the browser on the current device. It does NOT yet synchronise an account between iPhone, iPad and laptop. Secure cloud accounts require a backend service, authentication configuration, a database and privacy/security setup. The files include a cloud architecture note for the next infrastructure sprint.

VOICE SUPPORT
Text-to-speech generally works in modern browsers. Speech recognition support varies. On iPhone and iPad, browser support can differ between Safari versions. Quick command buttons remain available when speech recognition is unavailable.

HOW TO HOST ON GITHUB PAGES
1. Extract this ZIP.
2. Upload all files to the root of the GitHub repository.
3. Commit the files.
4. In repository Settings > Pages, publish from the main branch/root.
5. Open the generated GitHub Pages address on each device.
