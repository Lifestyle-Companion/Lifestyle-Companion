HEALTHY EATING — FOUNDER TRIAL ALPHA 0.3

PURPOSE
Alpha 0.3 is a major founder-trial upgrade. It tests whether Healthy Eating can work equally well:
- with a talking Companion
- with a quiet Companion
- with no Companion

HOW TO RUN
1. Download the ZIP.
2. Extract the complete folder.
3. Open the extracted folder.
4. Double-click index.html.
5. Use a current version of Microsoft Edge, Google Chrome, Firefox or Safari.

IMPORTANT
Keep all files together in the same extracted folder. Do not open index.html from inside the ZIP.

MAJOR FEATURES IN THIS TRIAL
- Language and regional-format selection at the start of setup
- Optional Companion mode
- Companion on/off control in Settings
- Spoken instructions when a Companion and spoken guidance are enabled
- Written guidance when the Companion is disabled
- Companion and user-name pronunciation preview fields
- Four visual theme previews: Garden, Coast, Outback and Classic
- Structured country, state/region, postcode, suburb/city and street-address fields
- Country-aware postcode checks and phone-number formatting
- Preferred-name field
- Cal instead of kcal
- Slow & steady, Recommended and Fast weight-loss-rate choices
- No Fasting Plan or Custom Fasting Days
- User-selected goal weight plus a separate recommended goal
- Optional staged future goals
- Whole-number display for calculated recommendations
- Functional Weight Check-in room
- Recommendation recalculation after a change of at least 1 kg or a changed goal
- Tappable Home joke, quote, tip and food-fact card

PROTOTYPE LIMITATIONS
- No real account is created on a server.
- Email delivery and the six-digit code are simulated in the app.
- Trial information is stored only in local browser storage on that device/browser.
- Passwords are not uploaded or used for real authentication.
- Live address auto-completion is not connected. The trial provides structured fields, formatting and validation only.
- Phone validation is a founder-trial approximation and is not a substitute for a production phone-number service.
- Device voices differ by browser and operating system. Some browsers require a tap before speech starts.
- Nutrition calculations are estimates for workflow testing, not medical advice.
- Trend-based recommendation changes are deliberately simple in this trial.
- Diary, Food Database and Graphs remain Coming Soon rooms.

CALCULATION NOTE
The prototype uses BMI screening, the Mifflin–St Jeor resting-energy equation when male or female is selected, an activity multiplier and simple starting macronutrient estimates. Calculated display values are rounded to clean whole numbers. Weight measurements may retain one decimal place. These rules require professional nutrition review before public release.

DATA MIGRATION
Where browser storage makes it available, Alpha 0.3 attempts to migrate an existing Alpha 0.2 trial profile. A fresh extracted folder may still behave as a new local app depending on browser security and how the file is opened.
